Getting Started
===============

.. _Installation:
.. toctree::
   :maxdepth: 0
   


MLPro is listed in the `Python Package Index (PyPI) <https://pypi.org/project/mlpro/>`_ and also in `Anaconda <https://anaconda.org/mlpro/mlpro/>`_ . MLPro can be installed by running

.. code-block:: bash

    pip install mlpro

or

.. code-block:: bash

    conda install -c mlpro mlpro


The easiest way to become familiar with the concepts and functions of MLPro is to browse through the numerous :ref:`example programs <target_appendix1>`.
